# Dress Slot Images

Place manually uploaded dress images here using these exact filenames:

- `dress-001-front.jpg`
- `dress-001-back.jpg`
- `dress-002-front.jpg`
- `dress-002-back.jpg`
- Continue up to `dress-100-front.jpg` and `dress-100-back.jpg`.

Each catalogue slot is hard-coded to these paths. The admin page edits only the text information for each slot.
